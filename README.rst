
=======================
Covid19
=======================

Coronavirus2019 Dashboard


Requirement
===========

:Python: 3.6
:Django: 3.0


Quick start
===========
1. プロジェクトのクローン::

    git clone https://github.com/naritotakizawa/django-torina-tutorial1

2. モデルのmigrate::

    python manage.py migrate

3. うごかす::

    python manage.py runserver
